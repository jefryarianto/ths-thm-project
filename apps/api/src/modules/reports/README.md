# Modul Reports — Laporan & Statistik

Laporan dan statistik data anggota, penilaian, dan dokumen.

## Endpoint

| Method | Path                           | Deskripsi                |
| ------ | ------------------------------ | ------------------------ |
| GET    | /api/reports/members           | Laporan anggota          |
| GET    | /api/reports/assessments       | Laporan penilaian        |
| GET    | /api/reports/dashboard         | Dashboard statistik      |
| GET    | /api/reports/export/:type      | Export laporan (PDF/XLS) |
