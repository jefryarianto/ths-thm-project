# Awards — Template Piagam Penghargaan

Template piagam penghargaan THS-THM.
Format: portrait A4.

## Variabel Template
- `recipientName` — Nama penerima
- `awardTitle` — Judul penghargaan
- `category` — Kategori (contoh: Akademik, Non-Akademik)
- `reason` — Alasan penghargaan
- `date` — Tanggal
- `signatoryName` — Nama penandatangan
- `signatoryRole` — Jabatan penandatangan
