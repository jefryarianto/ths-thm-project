# Certificates — Template Sertifikat

Template sertifikat kelulusan / penghargaan THS-THM.
Format: landscape A4.

## Variabel Template
- `recipientName` — Nama penerima
- `title` — Judul sertifikat
- `description` — Deskripsi penghargaan
- `date` — Tanggal penerbitan
- `signatoryName` — Nama penandatangan
- `signatoryRole` — Jabatan penandatangan
