# BRD — Business Requirement Document

## Tujuan Bisnis
Sistem manajemen THS-THM bertujuan untuk mendigitalisasi proses administrasi keanggotaan, penilaian, dan penerbitan dokumen.

## Stakeholder
- **Pengurus THS-THM** — Admin sistem
- **Pembina / Penilai** — Input penilaian
- **Anggota** — Akses profil & dokumen via mobile app
- **Calon Anggota** — Pendaftaran online

## KPI
- Waktu input penilaian: < 2 menit per anggota
- Generate dokumen: < 10 detik per dokumen
- Import data: support 1000+ baris CSV
