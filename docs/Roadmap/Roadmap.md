# Roadmap - THS-THM System Manajemen

## Fase 1 – Pilot Project Larantuka
- CRUD anggota dan calon anggota
- Import CSV anggota lama → status incomplete → notifikasi anggota
- Dashboard web & mobile sederhana
- Latihan rutin dan pendadaran → absensi & input nilai penguji
- Template dokumen `.jsx` untuk kartu anggota, sertifikat, piagam
- Notifikasi FCM untuk anggota & admin

## Fase 2 – Nasional
- Multi-keuskupan support
- Pendaftaran baru & klaim anggota
- Dokumen resmi & organisasi → role-based akses
- QR validasi dokumen fisik
- Laporan kegiatan, latihan, pendadaran

## Fase 3 – Scale Nasional
- Dashboard statistik lengkap untuk admin & superadmin
- Semua modul terintegrasi (iuran, surat masuk/keluar)
- Pengelolaan aspek & item penilaian secara terpusat
- Reporting & analytics

## Fase 4 – Opsional / Future Enhancements
- Gamifikasi
- Forum komunitas
- Mobile app full features
- Integrasi notifikasi tambahan & reminder manual
