# Roadmap — Timeline Pengembangan

## Fase 1: Foundation (Month 1)
- Setup monorepo, database schema, auth module (7 roles)
- Basic CRUD: members, candidates, users
- Role guards & district/region/branch scoping

## Fase 2: Core Modules (Month 2)
- Registrations, claims, assessments
- Trainings (absensi, evaluasi)
- Graduations (peserta, penugasan penguji)
- Documents generation (kartu, sertifikat, QR)

## Fase 3: Operations (Month 3)
- Letters (surat masuk/keluar)
- Org-documents (AD/ART, SK, dll)
- Dues (iuran, tunggakan)
- Activities (kegiatan, kehadiran)
- Notifications (FCM + scheduling)

## Fase 4: Frontend & Mobile (Month 4)
- Dashboard admin role-based (Next.js)
- CSV import/export dengan validasi
- Mobile app (kartu digital, QR scan, FCM)
- Preview dokumen & kartu anggota

## Fase 5: Polish & Launch (Month 5)
- Reports & analytics dashboard
- E2E testing, bug fixing
- UAT, deployment (Render)
- Documentation finalization
